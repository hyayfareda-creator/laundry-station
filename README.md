# نظام إدارة مغسلة الملابس
هذا المشروع عبارة عن نظام لإدارة طلبات المغسلة، يتكون من ثلاث واجهات رئيسية مترابطة عبر Firebase:

## مكونات النظام:
* **client.html**: واجهة العميل لإرسال طلبات الغسيل.
* **admin.html**: لوحة تحكم الإدارة لمتابعة الطلبات وتحديث حالتها.
* **driver.html**: لوحة تحكم السائق لتوصيل الطلبات وتحديث حالتها إلى "تم التوصيل".

## متطلبات التشغيل:
* المشروع يعتمد على Firebase Realtime Database.
* تأكد من ضبط قواعد البيانات (Firebase Rules) لتسمح بالقراءة والكتابة.
* تأكد من تحديث رابط قاعدة البيانات في كل ملف إذا لزم الأمر.
